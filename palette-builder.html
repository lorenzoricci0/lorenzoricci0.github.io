Come funziona
1
Estrai le texture dal modpack

Vai nella cartella del modpack → .minecraft/mods/, oppure apri il file .jar del mod come ZIP ed estrai la cartella assets/<modid>/textures/block/.

2
Carica i PNG qui sotto

Trascina tutti i PNG delle texture dei blocchi. Il nome del file diventa l'ID del blocco (es. stone.png → stone). Puoi rinominare i file prima di caricarli.

3
Scarica palette.json

Il tool calcola il colore medio di ogni texture e genera il file da caricare nel Viewer.
